#### 2.2.3 拍摄角度建议

```
        俯拍 45°（主要角度）
              │
              ▼
    ┌─────────────────────┐
    │                     │
    │    ESP32 开发板      │  ← 环形补光灯
    │                     │
    └─────────────────────┘
      ↗       ↑       ↖
  侧面30°   正面微距   侧面30°
```

- **主角度**：俯拍 45°，环形补光灯均匀照亮设备，无硬阴影
- **特写角度**：微距镜头正面拍摄芯片丝印，浅景深突出主体
- **动态角度**：相机在滑轨上缓慢横移，从左到右展示设备全貌
- **组装场景**：鸟笼正面中景，背景虚化展示养殖环境

#### 2.2.4 脚本要点

> **旁白**："系统硬件基于 ESP32-WROOM 双核处理器，主频 240MHz，集成 4MB Flash 与 PSRAM。图像采集采用 OV3660 摄像头，支持 300 万像素硬件 JPEG 编码输出。环境感知方面，搭载 SHT40 工业级温湿度传感器与 MQ135 空气质量传感器，可实时监测温度、湿度及氨气等有害气体浓度。设备端通过 FreeRTOS 多任务调度，实现采集与传输的高效并行处理。"
