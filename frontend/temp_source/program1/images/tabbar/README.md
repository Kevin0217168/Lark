# TabBar 图标说明

本目录需要放置以下图标文件（建议尺寸：81px × 81px）：

## 所需图标列表：

1. home.png - 首页监测图标（未选中）
2. home-active.png - 首页监测图标（选中）
3. ai.png - AI自检图标（未选中）
4. ai-active.png - AI自检图标（选中）
5. clinic.png - 线上问诊图标（未选中）
6. clinic-active.png - 线上问诊图标（选中）
7. profile.png - 个人中心图标（未选中）
8. profile-active.png - 个人中心图标（选中）

## 设计规范：

- 颜色：未选中状态使用灰色 (#999999)，选中状态使用主色调 (#2ecc71)
- 风格：建议使用线性图标风格
- 背景：透明背景的PNG格式

## 临时方案：

如果暂时没有图标文件，可以使用小程序默认的TabBar样式，暂时移除tabBar配置中的iconPath配置。
