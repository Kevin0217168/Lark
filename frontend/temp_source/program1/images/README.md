# 图标资源说明

本项目需要以下TabBar图标，请使用以下尺寸的图片资源：

## TabBar 图标要求
- 尺寸: 81px × 81px
- 格式: PNG (建议透明背景)
- 名称规范:
  - home.png (首页 - 未选中)
  - home-active.png (首页 - 选中)
  - ai.png (AI自检 - 未选中)
  - ai-active.png (AI自检 - 选中)
  - clinic.png (问诊 - 未选中)
  - clinic-active.png (问诊 - 选中)
  - profile.png (我的 - 未选中)
  - profile-active.png (我的 - 选中)

## 临时方案
在开发阶段，可以使用以下方式：

1. 使用微信小程序内置图标
2. 使用在线图标库生成
3. 使用纯CSS绘制图标
4. 使用FontAwesome等图标库

## 建议图标风格
- 简约扁平化设计
- 与主题色 #00B894 保持一致
- 清晰易懂，符合微信设计规范
